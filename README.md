# axinstagram

Instagram embed generator and media downloader. Transform Instagram links into embeddable URLs that work in Discord, Telegram, Slack, X/Twitter, and more.

**Live at [axinstagram.com](https://axinstagram.com)**

## Features

- **Embed any post** - Get URLs that display Instagram media in chat apps
- **Download media** - Save photos and videos directly
- **Profile lookup** - View profile info, posts, stories, and highlights
- **High quality** - Original resolution, no cropping
- **Multiple themes** - Light, dark, and 10+ color schemes
- **No login required** - Works with public posts out of the box

## Supported Platforms

Discord, Telegram, X / Twitter, Slack, WhatsApp, iMessage, Signal, LinkedIn, Mastodon, Matrix, Reddit, FB Messenger

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

## Deployment

```bash
bun run deploy
```

## Testing

```bash
bun run test
```

## Tech Stack

- [Cloudflare Workers](https://workers.cloudflare.com/) - Edge runtime
- [Bun](https://bun.sh/) - Package manager & runtime
